# Route Type Restriction

`MFRouteType` enum

```objc
typedef NS_ENUM(NSUInteger, MFRouteType) {
  MFRouteTypeMotorway,
  MFRouteTypeTrunk,
  MFRouteTypeFerry,
  MFRouteTypeBridge,
  MFRouteTypeTunnel
};
```

Định nghĩa các loại đường sử dụng cho việc giới hạn *([MFRouteRestriction](reference/route-restriction.md))* tìm kiếm, tính toán chỉ đường.
